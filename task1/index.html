React - что это?

React (иногда React.js или ReactJS) — JavaScript-библиотека с открытым исходным кодом для разработки пользовательских интерфейсов.

React разрабатывается и поддерживается Facebook, Instagram и сообществом отдельных разработчиков и корпораций.

React может использоваться для разработки одностраничных и мобильных приложений. Его цель — предоставить высокую скорость, простоту и масштабируемость. В качестве библиотеки для разработки пользовательских интерфейсов React часто используется с другими библиотеками, такими как MobX, Redux и GraphQL.

Как развивался?

React был создан Джорданом Валке, разработчиком программного обеспечения из Facebook. На него оказал влияние XHP — компонентный HTML-фреймворк для PHP. Впервые React использовался в новостной ленте Facebook в 2011 году и позже в ленте Instagram в 2012 году. Исходный код React открыт в мае 2013 года на конференции «JSConf US».

React Native анонсирован на конференции Facebook «React.js Conf» в феврале 2015 года, а исходный код открыт в марте 2015 года. Он позволяет разрабатывать нативные Android-, iOS- и UWP-приложения с использованием React.

18 апреля 2017 года Facebook анонсировал React Fiber, переписанную и оптимизированную версию React. React Fiber станет основой разработки всех будущих функций и улучшений.

Как изучить?

  • Первый вариант - изучить все самостоятельно. В данном случае важна последовательность изучения материала, актуальность изучаемых технологий (проверяйте самые версию изучаемой технологии) и качество подачи материала. Если вы вообще не знакомы с web-разработкой, то ваша последовательность изучения может быть следующей (в каждом пункте ссылка на материал):
    • Основы интернет-технологий
    • HTML5: структура web-страницы, основные теги
    • CSS3: классы, псевдоклассы и псевдоэлементы, grid и flex разметки
    • JavaScript: переменные и типы данных, функции, массивы, объекты, DOM-дерево, алгоритмы
    • Git: основные команды, подключение библиотек в проект
    • React: основы компонентного подхода, JSX, React Hooks, React Router
      • P/S: изучение технологий не ограничивается просмотром видео на YouTube. Обязательно просматривайте документацию по технологиям.
  • Второй вариант - записаться на очный или онлайн курс по web-разработке. Данный вариант более "безболезненный" в плане подбора материала для изучения (его уже подобрали за вас, правда, стоит поинтересоваться у преподавателей изучаемыми версиями), хоть и потребует финансового вложения. Вот некоторые организации, предлагающие обучение основам web-разработки
    • ЦДПО ИТМО
    • Авалон (Политех)
    • Яндекс Практикум
    • Coursera
